Introduces support for "Multiple top-level domains" and "Display Advertiser Support" into the Standard dashboard.

== Description ==

"Standard Advanced Google Analytics" introduces two new fields in to Standard "General" tab section that makes it easy to add support for multiple top-level domains and to display advertiser support.

== Installation ==

There are two ways to install WordPress plugins: via the WordPress Dashboard (recommended) and via FTP. 

Installing via the WordPress Dashboard is recommended because this process makes sure all the file and folder permissions are set correctly.

= Using the WordPress Dashboard =

1. Download the Standard Plugins: https://github.com/eightbit/standard-plugins/archive/master.zip
2. Unzip the "standard-plugins-master.zip" archive on your computer.
3. Locate the "standard-advanced-google-analytics" directory within the "standard-plugins-master" directory.
4. Rezip the "standard-advanced-google-analytics" directory.
5. Locate the "Plugins" menu in the WordPress admin.
6. In the "Plugins" menu, click on "Add New".
7. On the "Add New" page, click on the "Upload" link at the top of the page.
8. On the "Upload" page, locate the "standard-advanced-google-analytics.zip" archive (zip) on your computer (created in Step #4) and click "Install Now".
9. On the successful upload page, click "Activate".

= Using FTP =

1. Download the Standard Plugins: https://github.com/eightbit/standard-plugins/archive/master.zip
2. Unzip the "standard-plugins-master.zip" archive on your computer.
3. Locate the "standard-advanced-google-analytics" directory within the "standard-plugins-master" directory.
4. Connect to your server via FTP.
5. Upload the "standard-advanced-google-analytics" directory to the "/wp-content/plugins/" directory on your server.
6. In the WordPress Dashboard, navigate to the "Installed Plugins" page under the "Plugins" menu.
7. Locate "Standard Advanced Google Analytics" and click "Activate".
